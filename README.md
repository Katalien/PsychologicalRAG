# Структура репозитория:
- `data/` директория для датасета
- `data/articles/` полные тексты датасета
- `data/final_dataset.json` - json файл с датасетом
- `scrits/` директория для парсинга данных и сбора финального json в нужной структуре данных
- В ветке `development` и `finalize_dataset` предсавелны скрипты для тестирвоания  запуска пайплайн работы рага
    - `model.py` классы для раг системы
    - `test_pipleine.py` - тест скрипта
    - `faiss_index` - векторизация бд

# Структура данных
Данные офомлены в json формат следующего вида:
```
{
  "rows": [
    {
      "id": 1,   # id источника
      "name": "Как справиться с депрессией самостоятельно",    # вопрос
      "link": "https://medscannet.ru/blogmed/kak-vyjti-iz-depressii/",    # ссылка
      "date": "08-09-2025",    # дата создания статьи
      "category": "depression",   # категория
      "text": "<текст статьи>"  # текст статьи
    },
```

## Источник данных
- Статьи психологической помощи из открытых источников в интернете от психоогов/психологических журналов
- Научные статьи, доказывающие достоверность и эффективность представленных техник


### Статьи представлены по следующим категориям:
- депрессия
- тревожность
- выгорание
- панические атаки
- стресс/сильные эмоции
- категория для научных статей


## Процесс работы [IN-DEV]
- Данные структурированы в единый json
- Написан пайплайн для векторизации данных
- Векторное представление хранится в ветке `finilize_dataset` в директории `faiss_index/`
- Написаны классы для запуска ответа системы в заранее заданном формате ответа (model.py)
- Система протестирована на тестовых кейсах (test_pipeline.py)
